# ÉLAN Website Project Instructions

## Project
Build a premium minimalist website for ÉLAN, a women’s strength private club / boutique training studio.

## Tech stack
- Next.js
- React
- CSS Modules or SCSS
- No TypeScript unless explicitly requested
- Keep the project simple, clean and production-ready

## Brand direction
ÉLAN is not a loud commercial gym. It is a private, calm, refined strength club for women.

Core ideas:
- Private
- Focused
- Elevated
- Strength
- Grace
- Discipline
- Precision
- Limited membership
- Membership by application only

## Visual style
Use a minimalist luxury aesthetic:
- light stone / beige / warm grey background
- black or near-black typography
- thin borders
- editorial spacing
- calm, premium, quiet atmosphere
- avoid neon, aggressive gym visuals, heavy gradients, loud fitness clichés

Suggested colors:
- Background: #f3f0eb
- Card / stone: #d8d2ca
- Text: #1f1d1b
- Muted text: #6f6962
- Border: #bdb5ac
- Accent gold: #9c7a43

Suggested fonts:
- Headings: Cormorant Garamond, Playfair Display, Bodoni Moda
- Body: Inter, Montserrat, Satoshi or similar clean sans-serif

## Reference files
Read these before creating or editing the site:
- docs/brand-context.md
- docs/content.md
- docs/visual-reference.jpg

## Site structure
Create a one-page website with these sections:
1. Hero
2. Brand statement
3. What makes ÉLAN different
4. Membership options
5. The space
6. For whom
7. Application CTA / contact form

## Content tone
Use refined English copy.
Avoid generic gym language like:
- beast mode
- crush your goals
- no pain no gain
- fitness journey clichés

Preferred tone:
- elegant
- calm
- precise
- aspirational but not cheesy
- premium but not arrogant

## Development rules
- Keep components small and readable
- Use semantic HTML
- Make it responsive
- Prioritize mobile and desktop polish
- Use accessible contrast
- Do not over-engineer
- Do not add authentication, database or backend unless requested
- Do not invent complex CMS logic at this stage

## Done means
- The page runs locally
- Layout is responsive
- Content is implemented
- Visual direction follows the reference
- No console errors
